RLVE — Reinforcement Learning via Environment

The cascade mechanism raises zone severity when unserviced — making the task harder as the agent improves. A greedy agent that rescues the nearest zone first cannot exploit this: critical zones escalate faster and cut into critical_zone + severity rewards. The reward function is designed so the only winning strategy is genuine multi-zone coordination.
